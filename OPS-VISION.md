# Ops UI Vision

Ops UI is a small, adaptable interface for making operational work visible. The goal is to help a team see what needs attention, understand recent changes, and coordinate follow-up without relying on scattered private notes.

Public documentation should stay generic. Private deployments can customize domains, integrations, schedules, and internal workflows in local configuration and private docs.

## Goals

- Make operational status easy to scan
- Keep alerts and follow-ups organized
- Support safe integration with private services
- Keep public code and docs free of sensitive information
